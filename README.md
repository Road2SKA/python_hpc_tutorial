# python_hpc_tutorial
This repository contains scripts & jupyter notebooks with examples of acceleration &amp; parallelization in python. This material is adapted from several different resources:
* CSCS HPC python course by Dr. Rafael Sarmiento & Dr. Theofilos Manitaras: https://github.com/eth-cscs/PythonHPC
* Vectorization & Broadcasting tutorial by Ayoosh Kathuria: https://blog.paperspace.com/numpy-optimization-vectorization-and-broadcasting/
* CuPy user guide: https://docs.cupy.dev/en/stable/user_guide/basic.html
* mpi4py tutorial: https://mpi4py.readthedocs.io/en/stable/tutorial.html
* Distributed Parallel Programming Patterns using mpi4py, Libby Shoop, Macalester College: https://colab.research.google.com/drive/1yxusXcFQ9ea1bff4_q5iToMhGeQnmSwC?usp=sharing
* Dask array, parallelized numpy by Dask Developers: https://tutorial.dask.org/02_array.html
